# StackForge - AI-Powered Social Media Marketing Platform

Welcome to the official documentation for StackForge, a production-ready SaaS boilerplate built with Next.js 14, TypeScript, and Tailwind CSS. StackForge is designed to help developers build AI-powered social media marketing platforms quickly and efficiently.

## üõ†Ô∏è Technology Stack

StackForge leverages cutting-edge technologies to provide a robust foundation:

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript (strict mode)
- **Styling**: Tailwind CSS with custom configurations
- **UI Libraries**: Shadcn UI, Aceternity UI, Magic UI
- **State Management**: React Context API
- **Form Handling**: React Hook Form with Zod validation
- **Authentication**: Clerk
- **Database**: PostgreSQL via Drizzle ORM
- **Package Manager**: Bun
