# Going to Production

Ready to launch your StackForge application? This guide covers everything you need to know about deploying your application to production, including checklists, authentication setup, environment variables, and deployment options.

## Overview

Deploying a StackForge application to production requires careful attention to several key areas:

- **Production Checklist**: Ensure all critical items are configured properly
- **Authentication Setup**: Configure authentication for your production environment
- **Environment Variables**: Set up and manage your production environment variables
- **Deployment Options**: Choose the right deployment platform for your needs
- **Database Setup**: Configure your production PostgreSQL instance
